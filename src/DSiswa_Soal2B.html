Pembahasan
Buat program yang meminta pengguna memasukkan jumlah mahasiswa dan nilai mereka. Program kemudian menghitung rata-rata nilai dan mencetak hasilnya. Jika tidak ada mahasiswa (jumlah = 0), program harus menampilkan pesan yang sesuai. Pakai formatting untuk menunjukkan hasil nilai

Input Format
int jumlah mahasiswa, int setiap nilai

Constraints
jumlah : 1 < n < 100 nilai : 0 <=m <=100

Output Format
String yang menampilkan rata-rata mahasigma

Untuk dapat menghitung rata-rata, Anda dapat membuat scanner terlebih dahulu, lalu memikirkan bagaimana caranya array disimpan dan dihitung totalnya. Dengan menggunakan for loop, maka nilai total dapat dihitung
Cek Kode
Sample Input 0
Sample Output 0
5
80 100 89 20 30
Rata-rata : 63.80
import java.io.*;
import java.util.*;

public class Solution {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int n = scanner.nextInt();
scanner.nextLine();

int[] nilai = new int[n];

double total = 0;
for (int i = 0; i < n; i++) {
int tampung = scanner.nextInt();
nilai[i] = tampung;
total += nilai[i]; // Menambahkan nilai langsung ke total
}

// Menghitung rata-rata
double rataRata = total / n;
System.out.printf("Rata-rata : %.2f\n", rataRata);
}
}
