Array pada dasarnya adalah kumpulan variabel yang digabungkan menjadi 1.
Anda dapat menganggap array sebagai sebuah wadah yang menampung berbagai isi kotak didalamnya,
Di Java, Array dibuat dengan cara sebagai berikut.
Array memiliki lokasi variabel didalamnya yang disebut dengan index.
Index array selalu dimulai dari 0. Jika array belum diisi apapun, maka nilai default nya adalah null atau 0 tergantung tipe datanya.
Kita dapat inisialisasi nilai di array dengan berbagai cara, salah satunya adalah :
Berlaku juga untuk tipe variabel lainnya.
int[] listAngka = new int[5] //membuat array berisi 5 angka integer
int[] listAngka =[2, 3, 10, 222, 999, 1000] //membuat & mengisi nilai array dengan 5 angka integer
System.out.println(listAngka[1]); //mencetak angka 3
int angkaKetiga = listAngka[2]; //menyimpan nilai array kedalam variabel int biasa
listAngka[0] = listAngka[4]; //nilai 2 berubah menjadi 999
