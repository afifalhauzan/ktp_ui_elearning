Buat program untuk menerima input angka, lalu dimasukkan dalam array. Program akan mencetak 1 angka dari array tersebut yaitu pada index 5. Tugas anda adalah mengisi kode yang kosong agar program dapat menerima input berupa angka integer dan dimasukkan dalam array.

Input Format
int nilai angka

Constraints
nilai : 0 <= m <=1000

Output Format
Angka pada array pada index 5

Untuk dapat menerima input, program harus menggunakan Scanner terlebih dahulu, lalu menyimpan angka dalam array sesuai index pada perulangan.
import java.io.*;
import java.util.*;

public class Solution {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);

int n = scanner.nextInt();
int[] arr = new int[n];

for (int i = 0; i < n; i++) {
arr[i] = scanner.nextInt();
}

System.out.println(arr[5]);
